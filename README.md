# goit-js-hw-11 — Image Search

A Pixabay-powered image search gallery with SimpleLightbox and iziToast integration.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Search images via Pixabay API
- Responsive masonry-style gallery
- Lightbox modal via SimpleLightbox
- Toast notifications via iziToast
- CSS spinner loading indicator
- Error handling with catch()

## css-loader

A pure-CSS spinner is used in `src/css/loader.css`. It is shown before each request and hidden when the response arrives.
